    <div class="project-detail" id="detail-elemental">
      <h4>Modular GAS Plugin for Unreal Engine 5</h4>
      <p>A modular Gameplay Ability System (GAS) plugin for Unreal Engine 5 designed to streamline equipment management and complex elemental interactions.</p>
      <h5>Key Features</h5>
      <ul>
        <li><strong>Data-Driven Equipment System:</strong> Manage visuals, abilities, and elemental tags through <code>UEAEquipmentDefinition</code> Data Assets.</li>
        <li><strong>Configurable Elemental Reactions:</strong> Define dynamic rules (e.g., <code>Fire + Oil = Explosion</code>) via Project Settings, applied dynamically in the Attribute Set.</li>
        <li><strong>Ability Wizard:</strong> Custom editor tool to automate the creation and "auto-wiring" of Gameplay Abilities and Effects.</li>
        <li><strong>Automatic Lifecycle:</strong> Handles the granting and revoking of abilities and loose tags automatically during item equip/unequip.</li>
      </ul>
      <p><span class="tech-tag">Unreal Engine 5</span> <span class="tech-tag">C++</span> <span class="tech-tag">Gameplay Ability System (GAS)</span> <span class="tech-tag">Editor Scripting</span> <span class="tech-tag">Mermaid.js</span></p>
      <p><strong>Repository:</strong> <a href="https://github.com/hemalpatel2990/ElementalArsenal" target="_blank">ElementalArsenal on GitHub</a></p>
    </div>

    <div class="project-detail" id="detail-uds">
      <h4>Open Source Tool for Unreal Engine</h4>
      <p>A high-performance networking solution for Unreal Engine designed to solve the challenge of replicating arbitrary data structures without custom C++ code.</p>
      <h5>Key Features</h5>
      <ul>
        <li><strong>Dynamic Struct Replication:</strong> Replicate any <code>USTRUCT</code> marked as <code>BlueprintType</code> without modifying C++ source</li>
        <li><strong>Fast Array Serialization:</strong> Built on <code>FFastArraySerializer</code> for minimized bandwidth usage</li>
        <li><strong>Type Safety:</strong> Optional schema enforcement to restrict keys or values to specific types</li>
        <li><strong>Event-Driven:</strong> Dedicated events for key addition, updates, and removal for responsive UI</li>
      </ul>
      <p><span class="tech-tag">Unreal Engine</span> <span class="tech-tag">C++</span> <span class="tech-tag">Networking</span> <span class="tech-tag">Replication</span> <span class="tech-tag">FastArraySerializer</span></p>
      <p><strong>Repository:</strong> <a href="https://github.com/hemalpatel2990/UniversalDataSync" target="_blank">UniversalDataSync on GitHub</a></p>
    </div>

    <div class="project-detail" id="detail-neothreads">
      <h4>Lightweight Concurrency Framework</h4>
      <p>A lightweight concurrency framework designed for Unreal Engine to optimize CPU-intensive operations by offloading them from the Game Thread.</p>
      <h5>Key Features</h5>
      <ul>
        <li><strong>Off-Thread Processing:</strong> Efficiently offloads computationally expensive operations to prevent frame hitches.</li>
        <li><strong>Stability Improvement:</strong> Significantly improves frame stability and performance for complex systems.</li>
        <li><strong>Developer Friendly:</strong> Built using C++ and the Unreal Task Graph for seamless integration.</li>
      </ul>
      <p><span class="tech-tag">C++</span> <span class="tech-tag">Unreal Engine</span> <span class="tech-tag">Multithreading</span> <span class="tech-tag">Task Graph</span></p>
