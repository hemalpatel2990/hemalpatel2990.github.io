## Local Development

### Prerequisites
- Ruby (3.0+)
- Bundler
- Git

### Setup

```bash
# Clone the repository
git clone https://github.com/hemalpatel/hemalpatel.github.io.git
cd hemalpatel.github.io

# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# View at http://localhost:4000
```

## Customization

### Update Content
- **Homepage:** Edit `index.markdown`
- **About:** Edit `_pages/about.markdown`
- **Projects:** Edit `_pages/projects.markdown`
- **Contact:** Edit `_pages/contact.markdown`

### Update Styling
- **Custom CSS:** Edit `assets/css/custom.css`
- **Theme Settings:** Edit `_config.yml`

### Add Projects
Add project images to `assets/images/` and update project descriptions in `_pages/projects.markdown`

## Deployment

Site automatically deploys to GitHub Pages on push to `main` branch.

```bash
git add .
git commit -m "Update content"
git push origin main
```

Changes go live in 1-2 minutes.
